# windowfy
jquery plugin to create window-based interfaces

Use
===
```` 
$('element').windowfy(params)
/*
    title: 'Title',
    minimize: true,
    close: true,
    onClose: function() {},
    onMinimize: function() {},
    id: string
*/
````

Including
=========

The minified JS file is at /<installation>/bin/jquery.windowfy.min.js

The CSS file is at /<installation>/bin/jquery.windowfy.css
